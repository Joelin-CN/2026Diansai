# 2026电赛 —— 电源板设计方案

**版本**: 1.0
**日期**: 2026-07-11
**设计目标**: 基于 14.8V 4S 航模电池，为整车系统（RDK X5、4路TB6612电机驱动、2路舵机）提供统一供电

---

## 1. 系统概述

### 1.1 输入

| 参数 | 规格 |
|------|------|
| 电池类型 | 4S LiPo 航模电池 |
| 标称电压 | 14.8V |
| 电压范围 | 12.8V (3.2V/cell) ~ 16.8V (4.2V/cell) |
| 容量 | 5200mAh |
| 总能量 | ≈ 77Wh |

### 1.2 负载汇总

| 负载 | 电压 | 持续电流 | 峰值电流 | 备注 |
|------|------|----------|----------|------|
| 4×TB6612 电机驱动 | 12V (VM) | 4.8A (4×1.2A) | 12.8A (4×3.2A) | 实际同时峰值罕见，按 8A 设计 |
| 2×舵机 | 12V | 2A (合计) | 2A | — |
| RDK X5 | 5V | 5A | 5A | x86 SBC |
| STM32F103核心板 | 3.3V | — | — | 核心板自带LDO，5V输入 |
| 2×ICM42688 + OLED + 蓝牙 | 3.3V | — | — | STM32核心板供电 |

### 1.3 输出轨

| 电压轨 | 设计电流 | 设计功率 | 拓扑 |
|--------|----------|----------|------|
| 12V | 8A cont / 10A peak | 96W cont | 同步降压 (Buck) |
| 5V | 5A | 25W | 同步降压 (Buck) |

## 2. 供电树

```
BATTERY 14.8V (XT60 输入)
│
├─ F1 (15A 保险丝)
│
├─┬──────────────────────────────┐
│ │                              │
│ ▼                              ▼
│ BUCK_12V                      BUCK_5V
│ 14.8V → 12V                   14.8V → 5V
│ 8A cont / 10A peak            5A
│                              │
│ ▼                              ▼
│ 12V_BUS                      5V_BUS
│ │                              │
│ ├─ J12V_1 ─→ TB6612#1 VM      └─ J5V ─→ RDK X5 (5.5×2.1mm DC)
│ ├─ J12V_2 ─→ TB6612#2 VM
│ ├─ J12V_3 ─→ TB6612#3 VM
│ ├─ J12V_4 ─→ TB6612#4 VM
│ ├─ JSERVO1 ─→ 舵机#1
│ └─ JSERVO2 ─→ 舵机#2
│
├─ VBAT_SENSE ─→ MCU ADC (分压1:5, 电池电压监测)
│
└─ GND ── 整板共地 / 大面积铺铜
```

**关键设计决策**:
- 12V 与 5V **完全独立**，各自从电池取电，互不干扰
- STM32核心板和TB6612稳压模块自带3.3V LDO，电源板不提供3.3V
- 当需要使用 STM32 下载调试时，核心板通过 USB 供电即可（无需启动12V轨）

## 3. 功率预算与续航估算

### 3.1 典型工况

| 工况 | 12V负载 | 5V负载 | 总功率 | 预估续航 |
|------|---------|--------|--------|----------|
| 待机 | 2A (舵机保持) | 2A (RDK空闲) | 34W | ~2.3h |
| 低速行驶 | 4A (电机+舵机) | 3A (RDK视觉) | 63W | ~1.2h |
| 全速行驶 | 7A | 5A | 109W | ~42min |
| 极限峰值 | 10A | 5A | 145W | ~32min |

> 续航 = 77Wh × 0.85 (效率) / 负载功率

### 3.2 单轨详细预算

**12V 轨**:
| 负载 | 电流/路 | 路数 | 小计 |
|------|---------|------|------|
| TB6612 VM (持续) | 1.2A | 4 | 4.8A |
| TB6612 VM (峰值, 2路同时) | 3.2A | 2 | +4A |
| 舵机 | 1A | 2 | 2A |
| **合计** | **持续 6.8A / 峰值 10.8A** | | |

设计余量: 8A持续 / 10A峰值 (约15%~47%余量)

**5V 轨**:
| 负载 | 电流 | 
|------|------|
| RDK X5 (满载) | 5A |
| **合计** | **5A** |

设计余量: 5A持续 (按RDK规格满载设计，实际余量取决于RDK实时功耗)

## 4. 12V Buck 设计 (14.8V → 12V / 8A)

### 4.1 芯片选型

| 参数 | 要求 | 备注 |
|------|------|------|
| Vin 范围 | ≥ 16.8V + 裕量 | 考虑4S满电+瞬态 |
| Vout | 12V (±3%) | — |
| Iout | ≥ 8A cont, 10A peak | — |
| 拓扑 | 同步整流 | 高效率, 低发热 |
| 开关频率 | 300k~800kHz | 平衡效率与尺寸 |

**推荐方案**:

| 方案 | 芯片 | 类型 | Iout | 优点 | 缺点 |
|------|------|------|------|------|------|
| A (推荐) | LM25116 / LM5116 | 控制器+外置MOS | 可扩展至15A+ | 电流大, 灵活, 多源 | PCB布局要求高, 元件多 |
| B | TPS54824RNVR | 全集成功率管 | 8A | 电路简单, 元件少 | 输入18V裕量偏紧 |
| C | LMR51635 / AP64502 | 全集成功率管 | 5~6A | 便宜, 好买 | 电流不够, 需两颗并联 |

**最终推荐: 方案 A (LM25116)**，理由:
- 输入范围 6V~42V，4S 电池 16.8V 满电有余量
- 外置 MOSFET 可灵活选择 Rdson，轻松支撑 10A+
- 同步整流，峰值效率 > 95%
- 可调频率 (50kHz~1MHz)

### 4.2 LM25116 原理图参数

```
关键外围参数:
- Fsw: 400kHz (Rt ≈ 27.4kΩ)
- L: 10μH (饱和电流 ≥ 14A)
- Cin: 2× 22μF 25V X7R MLCC + 1× 100μF 25V 铝电解
- Cout: 3× 22μF 25V X7R MLCC + 1× 220μF 25V 铝电解
- 高侧MOS: N-MOS, Vds≥30V, Rdson<10mΩ, Qg<15nC (如 CSD18540KTT)
- 低侧MOS: N-MOS, Vds≥30V, Rdson<5mΩ, Qg<25nC  (如 CSD18537NKTT)
- 反馈电阻: Rtop=100kΩ, Rbot=10kΩ → Vout=12V
- SS电容: 100nF → 软启动 ~8ms
```

### 4.3 12V 输出保护

- 输出过压保护(OVP): 通过反馈电阻设置 (硬件钳位，可选)
- 输出过流保护(OCP): LM25116 内置逐周期电流限制
- 输出 TVS: SMBJ15A 防瞬态

## 5. 5V Buck 设计 (14.8V → 5V / 5A)

### 5.1 芯片选型

| 参数 | 要求 |
|------|------|
| Vin 范围 | ≥ 16.8V + 裕量 |
| Vout | 5V (±3%) |
| Iout | 5A cont |
| 拓扑 | 同步整流 |

**推荐方案**:

| 方案 | 芯片 | Iout | 优点 | 缺点 |
|------|------|------|------|------|
| A (推荐) | TPS565247 / TPS56637 | 5A~6A | 全集成功率管, 电路极简, 同步整流 | 输入最高18V (紧但够) |
| B | AP63205 / AP64502 | 3.5A~5A | 便宜好买 | 需确认输入范围 |
| C | LM25116 (复用) | 可扩展 | 与12V统一BOM | 成本高, 5A不值当用控制器 |

**最终推荐: 方案 A (TPS565247)，理由:**
- 4.5V~18V 输入，覆盖 4S 电池全范围
- 5A 连续输出，集成 33mΩ/19mΩ MOSFETs
- Eco-mode 轻载高效
- SOP-8 散热焊盘封装，布局友好
- 备选: TPS5450DDAR (5.5V~36V, 5A, 非同步但更宽输入)

### 5.2 TPS565247 原理图参数

```
关键外围参数:
- Fsw: 500kHz (内部固定)
- L: 4.7μH (饱和电流 ≥ 7A, DCR < 20mΩ)
- Cin: 2× 22μF 25V X7R MLCC
- Cout: 2× 47μF 10V X7R MLCC
- Cboot: 100nF
- 反馈电阻: Rtop=80.6kΩ, Rbot=20kΩ → Vout=5.0V
- EN分压: 电池电压直接使能，UVLO ≈ 6V (低于6V关闭)
```

### 5.3 5V 输出保护

- 输出 TVS: SMBJ6.5A
- TPS565247 内置 OCP / OTP / UVLO
- 输出端串联 5A PTC 自恢复保险丝（防RDK短路拉死整个5V轨）

## 6. 输入保护电路

### 6.1 总输入保护

```
BAT+ ──┬─ F1 (15A) ──┬─ TVS1 (SMBJ18A) ──┬─ C_IN_BULK ──┬─ 12V_Buck
       │              │                   │              │
       │              │                   │              └─ 5V_Buck
       │              │                   │
       │              └─ GND              └─ GND
       │
       └─ GND
```

| 元件 | 型号 | 参数 | 作用 |
|------|------|------|------|
| F1 | 汽车插片保险丝 | 15A / 32V | 过流/短路保护 |
| TVS1 | SMBJ18A | 18V 击穿, 600W | 防电池反接瞬态/浪涌 |
| C_IN_BULK | 2× 47μF 25V MLCC + 1× 220μF 35V 电解 | — | 输入滤波，抑制电池引线电感引起的振铃 |

### 6.2 防反接电路（可选）

如果不需要防反接（XT60 接口有防呆），可省略以降低损耗。

**如需要**，推荐方案:
```
BAT+ ──→ P-MOS (Vgs>20V, Rdson<5mΩ, e.g. IRF4905) → 负载
                    │
                    ├─ Rgs 100kΩ → BAT+
                    └─ 栅极 → 接 BAT- (通过电阻)
```
或简单方案: BAT+ → 肖特基二极管 → 负载（损耗大，15A×0.4V=6W，不推荐）

### 6.3 电池电压检测

```
BAT+ ── R1 (40kΩ) ──┬─ VBAT_SENSE ─→ MCU ADC
                     │
                     R2 (10kΩ)
                     │
                    GND
```

分压比 1:5，16.8V → 3.36V (MCU 3.3V ADC 安全范围)
低电量阈值: 14.0V (3.5V/cell) → ADC读数 2.8V

## 8. 热设计

### 8.1 损耗估算

| 轨 | 输出功率 | 效率估算 | 损耗 |
|----|----------|----------|------|
| 12V (8A) | 96W | ~93% | ~7.2W |
| 5V (5A) | 25W | ~90% | ~2.8W |
| **合计** | | | **~10W** |

### 8.2 散热措施

- **LM25116 (12V Buck)**: MOSFET 使用 DPAK/DFN 封装，底部大面积铜皮散热，必要时加铝散热片
- **TPS565247 (5V Buck)**: SOP-8 散热焊盘连接到底层 GND 铜皮，通过过孔阵列导热
- **电感**: 选用低 DCR (< 15mΩ) 型号，允许自身发热 < 40°C 温升
- **PCB**: 建议 2oz 铜厚，大电流路径开窗加锡

## 9. PCB 布局指南

### 9.1 层叠建议

**推荐 2 层板 (2oz 铜厚)**:
- 顶层: 功率走线 + 元件 + 局部 GND 铺铜
- 底层: 完整 GND 平面

如果铜厚只能 1oz，需加宽大电流走线并用开窗加锡补偿。

### 9.2 走线宽度

| 轨 | 电流 | 铜厚1oz 宽度 | 铜厚2oz 宽度 |
|----|------|------------|------------|
| BAT+ 总线 | 13A | ≥ 15mm | ≥ 8mm |
| 12V_BUS | 10A | ≥ 12mm | ≥ 6mm |
| 5V_BUS | 5A | ≥ 4mm | ≥ 2.5mm |
| 信号线 | < 100mA | 0.25mm | 0.25mm |

> 以上为 10°C 温升参考值

### 9.3 布局分区

```
┌──────────────────────────────────────────┐
│  XT60 电池输入 (左侧)                      │
│  ┌─────────┐   ┌─────────┐               │
│  │  保险丝  │   │  TVS    │               │
│  └────┬────┘   └────┬────┘               │
│       │  输入电容   │                     │
│       ├────────-────┤                    │
│       │             │                    │
│  ┌────▼────┐   ┌───▼─────┐               │
│  │ LM25116 │   │TPS565247│               │
│  │ 12V Buck│   │ 5V Buck │               │
│  │ (大面积)│   │ (右侧)   │               │
│  └────┬────┘   └───┬─────┘               │
│       │             │                    │
│  ┌────▼────┐   ┌───▼─────┐               │
│  │12V端子  │   │DC 5.5   │               │
│  │×4+舵机×2│   │插座     │               │
│  └─────────┘   └─────────┘               │
└──────────────────────────────────────────┘
```

**关键规则**:
1. 两个 Buck 的功率回路面积最小化 (Cin → 上管 → 电感 → Cout → GND)
2. 开关节点 (SW) 走线尽量短粗，避免与敏感信号平行
3. 反馈走线 (FB) 远离电感和 SW 节点
4. Buck 各自独立 GND 回流到输入电容负极，再星型汇合到电池 GND
5. 大电流走线开窗 + 加锡

## 12. 测试验证方案

### 12.1 上电前检查

1. 万用表二极管档测 BAT+ 与 GND 无短路
2. 测 12V_BUS 与 GND 无短路
3. 测 5V_BUS 与 GND 无短路
4. 目检焊接质量（虚焊、桥连）

### 12.2 空载测试

1. 接入可调电源 (设 14.8V/限流 1A)
2. 12V 轨输出应为 11.8~12.2V
3. 5V 轨输出应为 4.85~5.15V
4. 12V_LED 和 5V_LED 应点亮
5. 热像仪检查无异常热点

### 12.3 带载测试

| 测试项 | 条件 | 合格标准 |
|--------|------|----------|
| 12V 满载 | 8A 电子负载 | Vout ≥ 11.7V, 纹波 < 120mVpp |
| 12V 峰值 | 10A / 30s | 不触发保护, 温升 < 60°C |
| 5V 满载 | 5A 电子负载 | Vout ≥ 4.75V, 纹波 < 100mVpp |
| 双轨满载 | 12V@8A + 5V@5A / 30min | 所有指标正常, 无过热 |

### 12.4 保护验证

| 测试 | 方法 | 预期 |
|------|------|------|
| 过流保护 | 12V输出短路 | Buck关闭/打嗝, 电池端电流<保险丝额定 |
| TVS 钳位 | 示波器抓取上电瞬态 | 无超过 20V 的尖峰 |

## 13. 设计风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| LM25116 布局不当振荡 | 12V 纹波超标 | 严格遵循 datasheet layout guide，功率回路最小化 |
| 4S 满电 16.8V 接近 TPS565247 最大 18V | 可能损坏 | 输入端加 TVS 钳位；备选 TPS5450 (36V in) |
| 电机回馈能量反灌 12V 轨 | 12V 电压抬高 | 12V 输出加 TVS + 必要时加 TVS 或齐纳钳位 |
| RDK X5 峰值电流超 5A | 5V 跌出范围 | TPS565247 短时过载能力强（~6.5A峰值），必要时刻意选 6A 型号 |

## 14. 后续扩展建议

1. **电流检测**: 在 12V 和 5V 输出端串入 INA219 / INA226 模块，MCU 可实时监测功耗
2. **遥控开关**: 增加一路 MOSFET 电子开关 + 光耦隔离，可由 MCU 或遥控器控制各路输出通断
3. **USB 充电宝输入**: 增加一路 USB-C PD 诱骗 12V/15V 输入作为备用电源（调试时不接电池用）
4. **电池平衡充电接口**: 引出 4S 平衡头到面板，方便不拆电池直接充电
