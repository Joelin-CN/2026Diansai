# MSPM0G3507 扩展板设计规范

**项目**: 2026电赛TI
**版本**: 1.0
**日期**: 2026-07-10
**设计目标**: 为 MSPM0G3507 核心板设计扩展板，集成双 ICM42688 IMU、TB6612 电机驱动接口、舵机UART接口、OLED I2C接口

---

## 1. 系统架构

### 1.1 整体结构

```
┌─────────────────┐
│ MSPM0G3507      │
│ 核心板          │ 
│ (H1/H2 排针)    │
└────────┬────────┘
         │ P1/P2 母座
         │
┌────────┴────────┐
│   扩展板         │
│ • ICM42688 x2   │
│ • 电源管理       │
│ • 接口转接       │
└─┬──┬──┬──┬──┬──┘
  │  │  │  │  │
  │  │  │  │  └─→ J_I2C (OLED)
  │  │  │  └────→ J_UART1 (舵机)
  │  │  └───────→ J_TB6612_J6 (Motor C/D + Enc3/4)
  │  └──────────→ J_TB6612_J4 (Motor A/B + Enc1/2 + 3.3V输入)
  └─────────────→ ServoPWR (10V独立电源)
```

### 1.2 设计方案选择

**采用方案 A：单电源轨设计**

- 主 3.3V 电源来自 TB6612 驱动板 J4 的 3V3 输出
- 电源开关 SW1 控制整个扩展板（舵机电源除外）
- 10V 舵机电源独立输入，不受扩展板开关控制
- M0 排座 P1/P2 仅用于信号连接，不涉及电源供应

---

## 2. 电源设计

### 2.1 电源架构

**主电源轨 (3.3V_EXT):**

```
TB6612 J4-3V3 → SW1 (按键开关) → 3.3V_EXT → 扩展板所有器件
                                      ↓
                                 C_BULK (10μF)
                                      ↓
                                    LED1 + R_LED
```

**舵机电源 (10V):**

```
ServoPWR → J_UART1 Pin1/2 (直通，不经过SW1)
```

### 2.2 电源容量估算

| 器件 | 电流 |
|------|------|
| ICM42688 × 2 | ~6mA |
| OLED (0.96") | ~20mA |
| LED 指示灯 | ~3mA |
| **总计** | **< 30mA** |

TB6612 的 3.3V LDO 输出能力完全满足需求。

### 2.3 电源元件

| 位号 | 元件 | 规格 | 说明 |
|------|------|------|------|
| SW1 | 按键开关 | 红色 | 控制扩展板主电源 |
| LED1 | LED | 红色 0805 | 电源指示灯 |
| R_LED | 电阻 | 1kΩ 0603 | LED限流电阻 |
| C_BULK | 电容 | 10μF 0805 | 主电源去耦 |

### 2.4 PCB 布线要求

- **10V 舵机电源走线**: 宽度 ≥1.5mm（建议 2mm），减少发热
- **GND 铺铜**: 整板铺铜，所有 GND 连接到铺铜层
- **3.3V_EXT 走线**: 宽度 ≥0.5mm

## 3. 双 ICM42688 IMU 设计

### 3.1 元件

- **R48**: ICM-42688-P (IMU #0)
- **R49**: ICM-42688-P (IMU #1)

### 3.2 信号连接

两个 ICM42688 共享 SPI1 总线，通过独立片选区分：

| 信号 | MCU引脚 | 排座位置 | 连接方式 |
|------|---------|----------|----------|
| SPI1_SCLK | PA17 | P2 H2-12 | 并联到 R48/R49 SCLK |
| SPI1_PICO (MOSI) | PA18 | P2 H2-11 | 并联到 R48/R49 MOSI |
| SPI1_POCI (MISO) | PA16 | P2 H2-13 | 并联到 R48/R49 MISO |
| SPI1_CS0 | PB20 | P1 H1-8 | R48 CS (ICM0) |
| SPI1_CS1 | PA2 | P1 H1-6 | R49 CS (ICM1) |

**原理图网络标签:**
- R48: SPI_ICM0_SCLK, SPI_ICM0_MOSI, SPI_ICM0_MISO, SPI_ICM0_CS
- R49: SPI_ICM1_SCLK, SPI_ICM1_MOSI, SPI_ICM1_MISO, SPI_ICM1_CS
- 实际连接时，SCLK/MOSI/MISO 三根线连在一起（共享总线）

### 3.3 电源和去耦

每个 ICM42688:
- VDD → 3.3V_EXT
- GND → GND 铺铜
- C_ICM0/C_ICM1: 0.1μF (0402) 紧邻 VDD 引脚
- C_ICM0_B/C_ICM1_B: 10μF (0805) 可选，增强电源稳定性

### 3.4 未使用引脚

按数据手册处理，通常悬空：
- INT1, INT2 (驱动不需要中断)
- FSYNC
- CLKIN
- RESV 引脚

### 3.5 PCB 布局建议

- 两个 ICM42688 放置在远离电机驱动接口的位置（减少EMI）
- 去耦电容尽可能靠近芯片 VDD 引脚（< 5mm）
- SPI 信号线等长布线（如果可能）
- 避免在 ICM42688 下方走高频信号线

## 4. TB6612 电机驱动接口

### 4.1 连接器

- **J_TB6612_J4**: 2x6Pin PH2.54 双排母座（连接 Motor A/B）
- **J_TB6612_J6**: 2x6Pin PH2.54 双排母座（连接 Motor C/D）
- 使用 PH2.54 2x6 排线连接

### 4.2 J_TB6612_J4 引脚映射

| Pin | 左列信号 | 方向 | MCU信号 (排座) | | Pin | 右列信号 | 方向 | MCU信号 (排座) |
|-----|----------|------|----------------|---|-----|----------|------|----------------|
| 1 | 3V3_IN | ← | (电源输入) | | 1 | STBY | → | PA23 (P2 H2-6) |
| 2 | PWMB | → | PA22 (P2 H2-7) | | 2 | PWMA | → | PA21 (P2 H2-8) |
| 3 | BIN2 | → | PA7 (P1 H1-11) | | 3 | AIN2 | → | PB7 (P1 H1-17) |
| 4 | BIN1 | → | PB8 (P2 H2-10) | | 4 | AIN1 | → | PB6 (P1 H1-16) |
| 5 | E2A | ← | PA14 (P2 H2-15) | | 5 | E1A | ← | PA12 (P2 H2-17) |
| 6 | E2B | ← | PA26 (P2 H2-3) | | 6 | E1B | ← | PA13 (P2 H2-16) |

**方向说明:**
- `→` 从 MCU 输出到 TB6612 (控制信号: PWM, IN1/IN2, STBY)
- `←` 从 TB6612 输入到 MCU (编码器反馈: E1A/B, E2A/B)

### 4.3 J_TB6612_J6 引脚映射

| Pin | 左列信号 | 方向 | MCU信号 (排座) | | Pin | 右列信号 | 方向 | MCU信号 (排座) |
|-----|----------|------|----------------|---|-----|----------|------|----------------|
| 1 | ADC | - | NC (不连接) | | 1 | GND | - | GND |
| 2 | PWMD | → | PA25 (P2 H2-4) | | 2 | PWMC | → | PA24 (P2 H2-5) |
| 3 | DIN2 | → | PB24 (P1 H1-7) | | 3 | CIN2 | → | PB9 (P2 H2-9) |
| 4 | DIN1 | → | PB19 (P1 H1-9) | | 4 | CIN1 | → | PA15 (P2 H2-14) |
| 5 | E4A | ← | PA31 (P1 H1-4) | | 5 | E3A | ← | PA27 (P2 H2-2) |
| 6 | E4B | ← | PB18 (P1 H1-10) | | 6 | E3B | ← | PA28 (P1 H1-3) |

### 4.4 电机和编码器映射

| TB6612 电机 | MCU 电机编号 | 编码器信号 |
|-------------|--------------|------------|
| Motor A | M1 | ENC1_A/B |
| Motor B | M2 | ENC2_A/B |
| Motor C | M3 | ENC3_A/B |
| Motor D | M4 | ENC4_A/B |

## 5. 通信接口设计

### 5.1 UART1 舵机接口 (J_UART1)

**连接器:** B6B-PH-K-S (6Pin PH2.0 立式母座)

**引脚定义 (从上到下):**

| Pin | 信号 | 方向 | 连接 | 备注 |
|-----|------|------|------|------|
| 1 | 10V | ← | ServoPWR | 宽走线 ≥1.5mm |
| 2 | GND | - | GND | 宽走线 ≥1.5mm |
| 3 | NC | - | 不连接 | 禁用 |
| 4 | NC | - | 不连接 | 禁用 |
| 5 | servoRx | ← | PA8 (P1 H1-14) UART1_TX | MCU→舵机 |
| 6 | servoTx | → | PA9 (P1 H1-15) UART1_RX | 舵机→MCU |

**设计要点:**
- 10V 和 GND 走线宽度 ≥1.5mm (建议 2mm)
- ServoPWR 电源独立，不经过 SW1 开关
- UART 信号走线远离电源线，避免干扰

### 5.2 I2C OLED 接口 (J_I2C)

**连接器:** 1x4Pin 2.54mm 间距母座（或排针）

**引脚定义:**

| Pin | 信号 | 方向 | 连接 |
|-----|------|------|------|
| 1 | GND | - | GND |
| 2 | VCC | - | 3.3V_EXT |
| 3 | SDA | ←→ | PA0 (P1 H1-1) I2C0_SDA |
| 4 | SCL | ← | PA1 (P1 H1-2) I2C0_SCL |

**I2C 上拉电阻:**
- R_SDA: 4.7kΩ (0603) 从 SDA 到 3.3V_EXT （必需，PA0 需要外部上拉）
- R_SCL: 4.7kΩ (0603) 从 SCL 到 3.3V_EXT （可选，PA1 核心板已有上拉，但建议添加以增强驱动）

**PCB 布局:**
- J_I2C 放置在扩展板边缘，便于 OLED 模块连接
- I2C 走线尽量短，避免靠近 PWM 信号

## 8. PCB 设计指南

### 8.1 层叠结构

建议采用 2 层板：
- 顶层：信号层 + 元件
- 底层：GND 铺铜 + 部分信号

如需更好的 EMI 性能，可采用 4 层板。

### 8.2 走线要求

| 信号类型 | 走线宽度 | 备注 |
|----------|----------|------|
| 10V 舵机电源 | ≥1.5mm (建议 2mm) | 减少发热 |
| 3.3V_EXT | ≥0.5mm | 主电源轨 |
| GND | 尽可能宽 | 铺铜 |
| SPI 信号 | 0.2~0.3mm | 等长布线（如可能） |
| I2C 信号 | 0.2~0.3mm | 尽量短 |
| PWM 信号 | 0.2~0.3mm | 远离模拟信号 |
| 编码器信号 | 0.2~0.3mm | 差分布线（如可能） |

### 8.3 布局建议

**分区布局:**
```
┌─────────────────────────────┐
│  P1      ICM42688    P2     │ ← 顶部：M0接口 + IMU
│ (左侧)    R48 R49   (右侧)  │
│                             │
│  SW1  LED1                  │ ← 中部：电源管理
│                             │
│  J_I2C              J_UART1 │ ← 中部：通信接口
│                             │
│  J_TB6612_J4  J_TB6612_J6   │ ← 底部：电机驱动接口
└─────────────────────────────┘
```

**关键点:**
1. ICM42688 远离电机驱动接口（减少 EMI）
2. 电源管理电路靠近 J_TB6612_J4（3.3V 输入源）
3. 去耦电容紧邻各芯片电源引脚
4. 接口连接器放置在板边，便于接线

### 8.4 接地设计

- 底层完整 GND 铺铜
- 顶层空闲区域补充 GND 铺铜
- 通过多个过孔连接顶层和底层 GND
- 模拟部分（ICM42688）和数字部分单点接地（如空间允许）

## 9. 测试和验证

### 9.1 上电测试

1. **不连接 TB6612**，用万用表测量 J_TB6612_J4 Pin1-左 (3V3_IN) 和 GND 之间无短路
2. **连接 TB6612**，不按 SW1，测量扩展板 3.3V_EXT 应为 0V
3. **按下 SW1**，测量 3.3V_EXT 应为 3.3V，LED1 点亮
4. 测量 ICM42688 VDD 引脚电压应为 3.3V

### 9.2 通信测试

**SPI 测试（ICM42688）:**
1. 读取 R48 (CS = PB20) 的 WHO_AM_I 寄存器 (0x75)，应返回 0x47
2. 读取 R49 (CS = PA2) 的 WHO_AM_I 寄存器 (0x75)，应返回 0x47
3. 验证两个 ICM 可以独立通信

**I2C 测试（OLED）:**
1. 扫描 I2C 总线，应检测到 OLED 地址（通常 0x3C 或 0x3D）
2. 发送测试数据，验证显示正常

**UART 测试（舵机）:**
1. 发送舵机控制指令，观察舵机响应
2. 如需反馈，读取 UART1_RX 数据

### 9.3 TB6612 接口测试

1. 连接 TB6612 驱动板
2. 设置 PWM 和方向控制信号，观察电机运转
3. 读取编码器信号，验证反馈正常

## 10. 已知限制和注意事项

### 10.1 电源依赖

- 扩展板完全依赖 TB6612 的 3.3V 输出
- 如果 TB6612 未连接或其 LDO 故障，扩展板无法工作
- **建议**: 在调试时可以临时从外部 3.3V 电源供电（通过 J_TB6612_J4 Pin1-左注入）

### 10.2 EMI 考虑

- 电机驱动信号（PWM、方向控制）可能对 ICM42688 产生干扰
- **缓解措施**: 
  - ICM42688 远离电机驱动接口
  - 良好的接地设计
  - 必要时在 PWM 信号上串联小电阻（如 100Ω）

### 10.3 舵机电源

- 10V 舵机电源路径直通，不受扩展板开关控制
- **安全提示**: 在维护或调试时，需要单独断开 ServoPWR 输入

### 10.4 I2C 上拉

- 根据引脚文档，PA0 (SDA) 需要外部上拉，PA1 (SCL) 核心板已有上拉
- 如果 I2C 通信不稳定，可以调整上拉电阻值（2.2kΩ ~ 10kΩ）

## 11. 后续扩展建议

### 11.1 可选增强功能

1. **备用电源输入**: 添加独立 3.3V 输入接口（通过二极管 OR 与 TB6612 供电合并）
2. **ESD 保护**: 在外部接口（J_UART1, J_I2C）添加 TVS 二极管
3. **电流监测**: 在 3.3V_EXT 路径添加电流检测电路
4. **状态 LED**: 为各通信接口添加活动指示 LED

### 11.2 预留接口

- P1/P2 上未使用的引脚可以预留测试点
- 为后续功能扩展预留 PCB 空间
