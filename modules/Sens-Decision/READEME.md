# 感知决策层

## 层级划分

```text
层级划分
┌─────────────────────────────────────────────────────┐
│ 1. 传感器预处理层                                    │
│    编码器脉冲→轮速；IMU→去噪去零偏；红外→归一化阵列  │
│    统一时间戳、统一到 base_link 坐标系               │
└───────────────┬─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────────────────┐
│ 2. 状态估计层                                        │
│    EKF 融合编码器+IMU → (x,y,θ,v,ω) + 协方差 P      │
│    输出 VehicleState（带 localization_valid）        │
└───────────────┬─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────────────────────┐
│ 3. 感知层（红外）  ── 三路输出，分开走               │
│   ① 连续量：横向偏差 d、航向估计 Δθ                 │
│   ② 离散事件：路口/断线/弯道入口（给 FSM 和 s 重置）│
│   ③ 有效性标志：连续丢线计数（给降级逻辑）          │
└────┬──────────┬──────────────┬──────────────────────┘
     │①         │②             │③
     ↓          ↓              ↓
┌─────────────────────────────────────────────────────┐
│ 4. 行为决策层（FSM）                                 │
│    IDLE/巡线/接近弯道/丢线降级/停车…                 │
│    事件②触发状态切换 + s 重置 + 误差率估计           │
└───────────────┬─────────────────────────────────────┘
                ↓ (当前行为 + 当前位姿)
┌─────────────────────────────────────────────────────┐
│ 5. 轨迹层                                            │
│    预存路径点序列 + 曲率限速 + S曲线速度规划          │
│    输出 TrajectoryPoint（x,y,θ,v,ω,a,α,κ）          │
└───────────────┬─────────────────────────────────────┘
                ↓
            运动控制层（不再展开）
```
