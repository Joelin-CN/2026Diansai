# MCP23017 循迹输入扩展

MCP23017 通过 I2C0 扩展 12 路数字循迹输入，不新增 MCU。

| 引脚 | 连接 |
|---|---|
| VCC / GND | 3.3 V 外置电源 / 与 MSPM0、传感器共地 |
| SDA / SCL | PA0 / PA1，与 OLED 共用 I2C0 |
| A0 / A1 / A2 | 全接 GND，地址 `0x20` |
| RESET | 上拉至 3.3 V |
| GPA0~GPA7 | 循迹 S1~S8 |
| GPB0~GPB3 | 循迹 S9~S12 |

所有输入必须为 3.3 V 逻辑；5 V 循迹输出须先做电平转换或分压。
