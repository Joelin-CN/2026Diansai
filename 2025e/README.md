# 2025 E 题工作区

本目录采用与 `2017b`、`2023e` 一致的分层结构，当前实现视觉检测、
激光位置解算、自动瞄准仿真和树莓派实时测试链路。

## 目录说明

- `calibration/`：棋盘格和后续原始标定产物。
- `configs/`：题目约束、运行参数和硬件配置。
- `data/`：后续测试输入数据。
- `device/`：设备配置文件。
- `logs/`：运行日志。
- `outputs/`：预览图、视频、仿真结果等生成物。
- `scripts/test/`：相机标定、树莓派VNC预览和 PyBullet 仿真入口。
- `src/vision/`：黑框检测和目标跟踪。
- `src/coordinate/`：靶心三维定位、激光安装偏移补偿和角度解算。
- `src/control/`：与硬件协议解耦的搜索、瞄准、连续锁定、发射和超时状态机。
- `src/task/`：基础与发挥部分任务编排。
- `tests/`：按源码层级组织的单元测试。
- `docs/`：题目资料、设计文档与交接记录。

实时测试入口是 `scripts/test/test_pi_camera_live.py`，当前直接处理原始摄像头画面，
不执行整帧去畸变。
